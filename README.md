# Cascades Pub Meal Order Collection

A mobile-first static website for collecting meal orders from friends for a pub meal at the Cascades hotel.

## Features

- View available meals with descriptions and prices
- Select single or customizable meals
- Customize multi-variation meals (e.g., Snitzel with type, size, sides, and sauce options)
- Adjust order quantities
- Clear all orders
- Copy formatted orders to clipboard

## Setup

1. Clone the repository
2. Open `index.html` in a web browser

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Clipboard API
